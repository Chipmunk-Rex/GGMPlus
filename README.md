## ✨ 기능 소개

### 현재 제공 기능
- **📋 자동 출석체크**: 설정한 시간마다 자동으로 출석체크 수행
- **🔑 토큰 자동 관리**: 로그인 상태 자동 유지 및 토큰 갱신
- **🔔 알림 시스템**: 출석체크 결과를 데스크톱 알림으로 표시
- **📊 로그 관리**: 출석체크 이력 확인 및 관리

### 개발 예정 기능
- **🎮 게임조각 자동 파밍**: 게임조각을 자동으로 수집
- **🎲 확률 계산기**: 게임 내 확률 계산 도구

## 🔧 설치 방법

### Chrome Web Store에서 설치 (예정)
Chrome Web Store에 등록 예정입니다.

### 수동 설치 (개발자 모드)
1. 이 저장소를 다운로드하거나 클론합니다:
   ```bash
   git clone https://github.com/Chipmunk-Rex/GGMPlus.git
   ```

2. Chrome 브라우저에서 확장 프로그램 페이지로 이동합니다:
   - Chrome 주소창에 `chrome://extensions/` 입력
   - 또는 메뉴 → 도구 더보기 → 확장 프로그램

3. 오른쪽 상단의 **개발자 모드** 활성화

4. **압축해제된 확장 프로그램을 로드합니다** 클릭

5. 다운로드한 폴더 선택

6. 설치 완료! 🎉

## 📖 사용 방법

### 1. 초기 설정
1. 확장 프로그램 설치 후 GGM 사이트(https://ggm.gondr.net)에 접속
2. 로그인하면 자동으로 인증 토큰이 수집됩니다
3. 확장 프로그램 아이콘을 클릭하여 로그인 상태 확인

### 2. 출석체크
#### 수동 출석체크
1. 확장 프로그램 팝업 열기
2. **🚀 수동 출석체크** 버튼 클릭
3. 결과 확인

#### 자동 출석체크 설정
1. 확장 프로그램 팝업에서 **⚙️ 설정** 클릭
2. **실행 간격** 및 **첫 실행 대기 시간** 설정
   - 실행 간격: 출석체크 주기 (기본 60분)
   - 첫 실행 대기: 브라우저 시작 후 첫 출석체크 대기 시간 (기본 1분)
3. **💾 설정 저장** 클릭

### 3. 로그 확인
1. 설정 페이지에서 **📋 출석체크 로그** 섹션 확인
2. 출석체크 이력 조회 가능
3. **🔄 새로고침** 버튼으로 최신 로그 확인

## 🎯 기능 상세

### 자동 출석체크 시스템
- **주기적 실행**: 설정된 시간 간격으로 자동 출석체크 수행
- **토큰 자동 갱신**: 토큰 만료 시 자동으로 백그라운드에서 갱신
- **중복 체크 방지**: 이미 출석한 경우 불필요한 알림 표시 안 함
- **오류 처리**: 네트워크 오류 또는 인증 오류 시 적절한 메시지 표시

### 토큰 관리
- **자동 수집**: 사이트 방문 시 localStorage/sessionStorage에서 토큰 자동 수집
- **실시간 갱신**: Storage 변경 감지로 토큰 변경 시 즉시 업데이트
- **만료 관리**: 토큰 만료 시 자동 갱신 시도

### 알림 시스템
- 출석체크 성공/실패 시 데스크톱 알림 표시
- 주요 이벤트(설치, 로그인 필요 등) 알림

## 🔒 권한 설명

이 확장 프로그램이 요청하는 권한:
- **storage**: 설정 및 토큰 저장
- **alarms**: 주기적 출석체크 스케줄링
- **cookies**: XSRF 토큰 접근
- **notifications**: 데스크톱 알림 표시
- **tabs**: 토큰 자동 갱신을 위한 탭 관리
- **scripting**: Content Script 주입
- **host_permissions (ggm.gondr.net)**: GGM 사이트에서만 작동

## 🛠️ 기술 스택

- **Manifest Version**: 3
- **Background**: Service Worker
- **Content Scripts**: 토큰 수집 및 페이지 상호작용
- **UI**: HTML/CSS/JavaScript (바닐라)

## 🐛 문제 해결

### 출석체크가 작동하지 않아요
1. GGM 사이트에 로그인되어 있는지 확인
2. 확장 프로그램 팝업에서 로그인 상태 확인
3. 로그인 상태가 아니라면 **🔑 로그인하러 가기** 클릭
4. 로그인 후 페이지 새로고침

### 토큰이 수집되지 않아요
1. GGM 사이트에서 로그아웃 후 다시 로그인
2. 페이지 새로고침 (Ctrl+F5 또는 Cmd+Shift+R)
3. 확장 프로그램 재설치

### 알림이 표시되지 않아요
1. Chrome 설정 → 개인정보 보호 및 보안 → 사이트 설정 → 알림
2. 알림이 차단되지 않았는지 확인
3. 확장 프로그램에 알림 권한이 부여되었는지 확인

### 전체 초기화하고 싶어요
1. 설정 페이지 열기
2. **🗑️ 데이터 관리** 섹션에서 **전체 초기화** 클릭
3. 모든 설정 및 로그가 삭제됩니다
