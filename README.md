## 기능 소개

### 현재 제공 기능
- **자동 출석체크**: 설정한 시간마다 자동으로 출석체크 수행
- **토큰 자동 관리**: 로그인 상태 자동 유지 및 토큰 갱신
- **알림 시스템**: 출석체크 결과를 알림으로 표시
- **로그 관리**: 출석체크 이력 확인 및 관리
- **빠른 이동 런처**: 자주 쓰는 GGM 화면을 팝업에서 바로 열기
- **글/댓글 임시저장**: 글쓰기와 댓글 입력 내용을 로컬에 자동 백업
- **새 글 알림**: 관심 게시판과 관심 동아리 게시글 변화 감지
- **골드박스/퀘스트 리마인더**: 확인할 타운 항목을 주기적으로 알림
- **주식 관심 알림**: 관심 동아리 주식 정보 변화 감지
- **활동 타임라인**: 출석, 알림, 설정 변경 기록을 로컬에 저장

## 설치 방법

### Chrome 수동 설치
1. 이 저장소를 다운로드하거나 클론합니다.
2. `chrome://extensions/`로 이동합니다.
3. 오른쪽 상단의 **개발자 모드**를 활성화합니다.
4. **압축해제된 확장 프로그램을 로드합니다**를 클릭합니다.
5. 이 저장소의 `chrome/` 폴더를 선택합니다.

### Firefox 수동 설치
1. 이 저장소를 다운로드하거나 클론합니다.
2. `about:debugging#/runtime/this-firefox`로 이동합니다.
3. **Load Temporary Add-on...**을 클릭합니다.
4. 이 저장소의 `firefox/manifest.json`을 선택합니다.

## 사용 방법

### 1. 초기 설정
1. 확장 프로그램 설치 후 [GGM 사이트](https://ggm.gondr.net)에 접속합니다.
2. 로그인하면 자동으로 인증 토큰이 수집됩니다.
3. 확장 프로그램 아이콘을 클릭하여 로그인 상태를 확인합니다.

### 2. 출석체크
#### 수동 출석체크
1. 확장 프로그램 팝업을 엽니다.
2. **수동 출석체크** 버튼을 클릭합니다.
3. 결과를 확인합니다.

#### 자동 출석체크
- 브라우저 시작 후 자동으로 출석체크를 시도합니다.
- 알람 기반으로 주기적인 출석체크를 수행합니다.

### 3. 로그 확인
1. 설정 페이지에서 **출석체크 로그** 섹션을 확인합니다.
2. 출석체크 이력을 조회할 수 있습니다.
3. **새로고침** 버튼으로 최신 로그를 확인합니다.

## 기능 상세

### 자동 출석체크 시스템
- **주기적 실행**: 설정된 시간 간격으로 자동 출석체크 수행
- **토큰 자동 갱신**: 토큰 만료 시 자동으로 백그라운드에서 갱신
- **중복 체크 방지**: 이미 출석한 경우 중복 시도를 방지
- **오류 처리**: 네트워크 오류 또는 인증 오류 시 적절한 메시지 표시

### 토큰 관리
- **자동 수집**: 사이트 방문 시 localStorage/sessionStorage에서 토큰 자동 수집
- **실시간 갱신**: Storage 변경 감지로 토큰 변경 시 즉시 업데이트
- **만료 관리**: 토큰 만료 시 자동 갱신 시도

### 알림 시스템
- 출석체크 성공/실패 시 알림 표시
- 주요 이벤트(설치, 로그인 필요 등) 알림

### 빠른 이동 런처
- 자유게시판, 퀘스트, 상점, 주식시장, 동아리, 프로젝트, 졸업작품 등 주요 화면으로 즉시 이동
- 확장 프로그램 팝업에서 새 탭으로 열기

### 글/댓글 임시저장
- 글쓰기, 게시글 상세 댓글, 동아리/프로젝트 게시판 입력창을 자동 감지
- 입력 내용은 브라우저 로컬 저장소에만 보관
- 입력칸이 비어 있을 때 저장된 초안 복구/삭제 버튼 표시

### 관심 알림
- 관심 게시판 카테고리와 동아리 게시글 ID를 설정해 새 글을 감지
- 골드박스와 퀘스트 상태를 주기적으로 확인
- 관심 동아리 주식 ID를 설정해 주식 정보 변화 알림
- 첫 확인은 기준값을 저장하고, 이후 변화부터 알림

## 권한 설명

이 확장 프로그램이 요청하는 권한:
- **storage**: 설정 및 토큰 저장
- **alarms**: 주기적 출석체크 스케줄링
- **cookies**: XSRF 토큰 접근
- **notifications**: 알림 표시
- **tabs**: 토큰 자동 갱신을 위한 탭 관리
- **host_permissions (ggm.gondr.net)**: GGM 사이트에서만 작동

## 문제 해결

### 출석체크가 작동하지 않아요
1. GGM 사이트에 로그인되어 있는지 확인합니다.
2. 확장 프로그램 팝업에서 로그인 상태를 확인합니다.
3. 로그인 상태가 아니라면 로그인 페이지로 이동합니다.
4. 로그인 후 페이지를 새로고침합니다.

### 토큰이 수집되지 않아요
1. GGM 사이트에서 로그아웃 후 다시 로그인합니다.
2. 페이지를 새로고침합니다.
3. 확장 프로그램을 다시 로드합니다.

### Firefox에서 임시 로드 테스트를 하고 싶어요
1. `about:debugging#/runtime/this-firefox`에서 `firefox/manifest.json`을 직접 선택합니다.
2. 임시 로드 테스트는 zip보다 `manifest.json` 직접 로드가 더 안정적일 수 있습니다.
